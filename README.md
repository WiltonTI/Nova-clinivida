Projeto da clínica do Dr Carlos Vinícius
🏥 Clinivida | Cuidado e Tecnologia
Landing page institucional da Clinivida, uma clínica multiespecialista focada em atendimento humanizado. O projeto apresenta as especialidades médicas, localização estratégica e um sistema de agendamento via modal.

🚀 Tecnologias Utilizadas
HTML5: Estrutura semântica para melhor acessibilidade e SEO.

CSS3: Estilização moderna utilizando variáveis (:root), Flexbox e CSS Grid.

JavaScript (ES6+): Lógica para abertura de modais e manipulação de agendamentos.

Google Fonts: Utilização da tipografia Plus Jakarta Sans.

Integrações: Google Maps API (Iframe) e WhatsApp Web.

✨ Funcionalidades
Design Responsivo: Adaptado para dispositivos móveis, tablets e desktops.

Hero Section: Apresentação de impacto com botões de chamada para ação (CTA).

Grade de Especialidades: Exibição interativa das áreas de atendimento (Cardiologia, Pediatria, etc.).

Agendamento Online: Formulário em modal para captação de leads e pacientes.

Rodapé Informativo: Endereço completo com mapa interativo incorporado.

📁 Estrutura de Arquivos
Plaintext
├── index.html      # Estrutura principal da página
├── styles.css      # Estilização e layout (CSS)
├── script.js       # Comportamento do modal e formulário
└── img/            # Diretório de imagens e logotipo

🛠️ Como Executar o Projeto
Clone este repositório ou baixe os arquivos.

Abra o arquivo index.html em qualquer navegador moderno.

Para visualizar as alterações de estilo, edite o arquivo styles.css.
